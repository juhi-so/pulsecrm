# PulseCRM

AI-ready CRM SaaS built with Next.js, Supabase, Tailwind CSS, n8n, and Vercel/Railway.

## Overview

PulseCRM is a modern full-stack CRM platform designed to help teams capture leads, manage sales pipelines, automate workflows, and track customer interactions.

The project includes:

* Landing page with lead capture form
* Admin dashboard
* Kanban sales pipeline
* Lead analytics
* Authentication system
* Supabase backend integration
* Workflow automation support with n8n
* Responsive SaaS UI
* Dark/light mode support

# Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

# Installation

Clone the repository:

```bash
git clone https://github.com/juhi-so/pulsecrm.git
```

Go into project:

```bash
cd pulsecrm
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

---

# Supabase Setup

Create a `leads` table:

```sql
create table leads (
  id bigint generated by default as identity primary key,
  name text,
  email text,
  company text,
  status text default 'New Lead',
  notes text,
  follow_up_date text,
  priority text default 'Medium',
  created_at timestamp with time zone default timezone('utc'::text, now())
);
```
